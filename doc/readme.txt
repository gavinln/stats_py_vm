This project demonstrates the following

1. To use this project in the root type
vagrant up

2. Connect to the VM
vm\ssh_vagrant.bat
or
python vm\ssh_vagrant.py
or on Linux
vagrant ssh

4. Start the notebook
/vagrant/vm/ipython_notebook.sh

5. Open the notebook in the browser at
http://localhost:8888/
